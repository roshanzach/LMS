1. Super Admin

manages the complete LMS platform and all registered colleges.

| Module              | Definition                                                |
| ------------------- | --------------------------------------------------------- |
| College Management  | Add, edit, or remove colleges using the LMS.              |
| Admin Management    | Manage College Admin accounts.                            |
| User Management     | Monitor user accounts across colleges.  		              |
| Notifications       | Send global announcements to colleges.                    |
| Reports & Analytics | View usage statistics and reports.   		                  |


2. College Admin

manages all academic and administrative operations within a college.

College Admin
│
├── Departments
├── Faculty
├── Students
├── Courses
├── Academic Calendar
└── Reports

| Module                | Definition                                                         |
| --------------------- | ------------------------------------------------------------------ |
| Department Management | Create and manage academic departments.                            |
| Faculty Management    | Add, update, and assign faculty members. (Assign - Positions)      |
| Student Management    | Manage student profiles and enrollments.                           |
| Semester Management   | Manage academic semesters and batches.                             |
| Course Management     | Create and assign courses for each semester.                       |
| Academic Calendar     | Publish important academic schedules and events.                   |
| Reports               | Generate academic and administrative reports.                      |
| Announcements         | Share notices with faculty and students.                           |
  


3. HOD

HOD

| Module                | Definition                                             |
| --------------------- | ------------------------------------------------------ |

| Subject Allocation    | Assign subjects to faculty members.                    |
| Attendance Monitoring | Monitor attendance of students in the department.      |
| Assessment Review     | Review internal marks and assessment progress.         |
| CO-PO Mapping         | Monitor course outcome and program outcome attainment. |

| Announcements         | Publish department-specific notices.                   |
